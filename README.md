## 🧠 What is Mukeem AI?

Mukeem AI is a minimal Transformer architecture implemented entirely from scratch using only NumPy.

This project demonstrates:

- Self-Attention (Q, K, V)
- Feedforward Neural Network
- Softmax + Cross Entropy Loss
- Full Backpropagation Through Attention
- Embedding Training
- Gradient Clipping
- Temperature-Based Text Generation

All running on a mobile device.

## 🧮 Mathematical Implementation

The model includes:

- Scaled Dot-Product Attention  
- Jacobian-based Softmax Backward Pass  
- Matrix Chain-Rule Backpropagation  
- Attention Weight Gradient Updates  
- Embedding Gradient Updates  
- Gradient Clipping for Stability  

Everything is implemented manually.

No automatic differentiation.
